## Detailed Description

### [1] README.md/

这是一个面向前端源码阅读、知识检索与技术复盘的索引型目录，重点围绕“Ctrl+F 快速定位”与 Wiki 式知识组织展开。内容覆盖 Vue2/Vue3、React Fiber、Scheduler、Hooks、响应式原理、事件循环、CSS/HTML、Webpack/Vite、Node.js、设计模式、架构设计，以及 RAG/Agent 等主题，整体更像一份前端技术知识地图或面试复习提纲。它适合希望快速查找概念、回顾原理、梳理源码阅读路径、沉淀工程经验的开发者，也适合用于学习笔记整理、知识库搭建和技术复盘。

### [2] AGENTS.md/

AGENTS 是 GitNexus Code Intelligence 的项目操作与工作流指导目录，强调在仓库中进行安全、可控的变更。其核心关注点包括：影响分析、执行流程审查、变更范围（blast radius）意识、修改前的差异确认、以及提交安全性检查。它更偏向“如何在仓库里做事”，而不是具体业务知识，适合需要遵循规范流程进行代码审查、修复、重构、提交和协作的开发者、自动化代理或代码助手。

### [3] CLAUDE.md/

该目录包含面向 GitNexus 索引实验室代码库的 `CLAUDE.md` 项目指南，重点是“安全地在仓库内工作”。内容围绕仓库导航、基于影响面的代码修改、符号级重构、调试与验证流程展开，强调在复杂代码库中保持变更可追踪、可验证、低风险。它适合需要借助规范化指引进行代码理解、定位、改动和回归验证的开发者，也适合用于大型仓库中的 AI 辅助开发场景。
