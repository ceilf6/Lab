## Detailed Description

### core.md `core.md`

这份文档是一篇关于 **HTML 图片元素用法** 的简明教程/备忘笔记，适合快速入门和查阅。内容分为多个部分：

- **属性**：介绍 `img` 的基础属性，重点说明 `src` 用于指定图片资源地址，`alt` 用于图片无法加载时的替代文本，也有助于无障碍访问与 SEO。
- **和 a 元素连用**：说明如何将图片包裹在 `a` 标签中，使图片本身成为可点击链接，实现点击图片跳转页面的效果。
- **map 连用**：讲解 image map 的基本概念，包括通过 `map` 的 `name` 与 `img` 的 `usemap` 进行关联，再用 `area` 按 `circle`、`rect`、`poly` 等 `shape` 划分可点击区域，并通过 `href` 分别跳转到不同目标；同时提到坐标原点与方向规则。
- **figure 元素连用**：说明 `figure` 常用于包裹图片、标题和描述，帮助构建更语义化、结构更清晰的内容块。

整体来看，这是一份偏实战、偏速查的 HTML 图片语义与交互笔记，尤其适合前端初学者和需要复习相关标签写法的人。
