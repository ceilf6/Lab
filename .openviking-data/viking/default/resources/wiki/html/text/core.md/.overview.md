## Detailed Description

### core.md `core.md`

这份文档是一个 HTML5 基础教学笔记，主要以“文本相关元素”为主线，帮助读者建立对 HTML 内容结构与 CSS 呈现分工的基本认识。文档采用标题层级组织内容，适合从最基础的语义标签开始逐步学习，并通过示意图与外部资源链接辅助理解。

文档重点介绍了多种常见标签的语义与表现特征：

- `h1 ~ h6`：用于定义不同级别的标题，体现文档层次结构。
- `p`：表示段落，是最常见的文本块结构之一。
- `span`：无语义的行内容器，常用于包裹局部文本并配合 CSS 做样式控制。
- `pre`：用于预格式化文本，保留空白与换行，适合展示代码或需要保持原始排版的内容。

此外，文档还解释了 HTML 中的空白折叠规则，以及“块级元素 / 行级元素”这类传统概念的来源与实际意义，帮助读者理解为什么不同标签在浏览器中的默认显示方式不同。对 `pre` 的说明尤其强调：它的保留格式效果并非标签本身“神奇实现”，而是依赖浏览器默认 CSS 中的 `white-space: pre`。这有助于读者将 HTML 语义与 CSS 样式机制联系起来。

整体来看，这是一份偏入门导向的前端基础资料，既适合学习 HTML 结构、语义化写法和文本排版，也适合想掌握代码块展示、空白控制、默认样式机制的初学者参考。
