## Detailed Description

### core.md `core.md`

这是一份介绍 HTML 列表标签的入门说明文档，重点讲解有序列表和无序列表的基本结构与使用方式。文档首先说明有序列表（ordered list）的写法：`ol` 作为父元素，`li` 作为子元素，列表中的每一项都必须放在 `li` 标签内，不能直接把其他元素作为 `ol` 的子节点。与此同时，文档介绍了 `type` 属性的作用，它可以改变默认的序号标识形式，例如数字、字母或其他样式，从而影响列表的展示效果。

在无序列表部分，文档说明只需将 `ol` 改为 `ul`，即可创建 unordered list。无序列表同样依赖 `li` 作为列表项容器，但其默认显示方式通常不是数字序号，而是项目符号。文档还特别强调，`ol` 的子元素只能是 `li`，这体现了 HTML 结构的语义化要求。

此外，文档提出一个实用建议：除非“序号”本身具有重要语义，否则更适合使用 CSS 来控制列表样式，而不是过度依赖 `type` 等 HTML 属性。这说明文档不仅讲“怎么写”，也讲“为什么这样写更合适”。整体内容非常适合前端初学者、HTML 自学者，以及需要快速回顾列表相关标签、属性和语义规范的读者。
