## Detailed Description

### core.md `core.md`

这是一份围绕 **HTML5 媒体元素** 的简明教程/笔记，内容按“`video`”“`audio` 音频”“兼容性”三部分展开，系统说明了网页中播放视频与音频的基础方法。

在 **`video`** 部分，文档重点介绍了 `controls`、`autoplay`、`muted`、`loop` 等常见属性的作用与使用规则，并强调这些多为 **布尔属性**：只要属性存在即可生效，不需要额外赋值。比如 `controls` 用于显示播放控制条，`autoplay` 用于自动播放，`muted` 用于静音播放，`loop` 用于循环播放。此部分适合帮助读者建立对 HTML5 视频标签的基本认知。

在 **`audio` 音频** 部分，文档说明 `audio` 的用法与 `video` 基本一致，也支持类似的属性和播放控制方式。也就是说，如果已经理解了 `video` 的结构，那么 `audio` 的掌握门槛会比较低，主要差异在于播放内容从视频转为音频资源。

在 **兼容性** 部分，文档强调 HTML5 新增的音视频能力在不同浏览器中支持的格式可能不一致，因此需要做好兼容处理。常见做法是在 `video` 中嵌套多个 `source` 标签，分别提供 `mp4`、`webm` 等候选资源，让浏览器按自身支持情况选择可播放的格式；同时还可以加入 **降级提示文本**，确保在不支持 HTML5 媒体播放的环境中，用户仍能看到替代信息。这个部分对于实际开发尤其重要，因为它直接关系到网页在不同终端和浏览器中的可用性。

总体来看，`core.md` 是一份以 **HTML5 video / audio / source / browser compatibility** 为关键词的入门型资料，适合前端初学者快速掌握媒体标签的核心概念，也适合开发者在项目中回顾媒体播放与兼容策略。
