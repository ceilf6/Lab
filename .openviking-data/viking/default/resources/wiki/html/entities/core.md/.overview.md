## Detailed Description

### core.md core.md

本文是一份 **HTML 实体字符（HTML Entity）** 的入门级参考说明，主要面向需要在网页中输出特殊符号的读者。文档先介绍实体字符的基本形式，分别是基于名称的写法 `&单词;`，以及基于数字编码的写法 `&#数字;`，帮助读者理解它们在 HTML 中的转义作用。随后列出了一些常用实体示例，包括小于号 `&lt;`、大于号 `&gt;`、空格 `&nbsp;`、non-breaking space，以及版权符号 `&copy;`，便于开发者在实际编写页面内容时直接查阅。

从用途上看，这份文档强调实体字符相当于代码中的转义处理机制：当某些字符在 HTML 中具有特殊语义时，可以使用实体写法来确保它们被正确显示，而不是被浏览器当作标签或语法解析。整体内容简洁、实用，适合作为前端开发中的速查资料，也适合 HTML 学习者建立对字符实体、字符编码和网页输出控制的基础认识。
