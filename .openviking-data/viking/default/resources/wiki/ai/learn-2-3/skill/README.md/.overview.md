# README

这份目录主要是一篇关于 **Claude Code 技能调用机制** 的说明性笔记，围绕 **用户输入、系统提示词（system prompt）、模型决策、JSON 工具调用、技能内容读取** 之间的完整交互流程展开。内容重点解释了初始化请求中的 **Tools / skill 定义**、**CLAUDE.md 与 system-reminder 的拼接方式**、以及模型何时决定调用本地技能、何时返回 UI 的流程关系。它还讨论了 **“渐进式披露”** 的技能读取策略、技能名称与调用格式要求，以及系统提示中如何向模型声明可用技能与触发条件。适合研究 **Claude Code 源码、AI agent 工具调用、prompt engineering、本地技能系统设计** 的开发者和技术读者。

## Detailed Description

### README.md README.md

这是一份聚焦 **Claude Code 技能调用机制** 的技术笔记，核心内容是梳理从用户输入进入系统，到模型判断是否调用技能，再到技能内容逐步读取和执行的完整链路。文档重点解释了初始化阶段中 **Tools / skill 的声明方式**，以及系统提示词中如何拼接 **CLAUDE.md** 和 **system-reminder**，让模型明确可用能力与触发条件。

文档还深入说明了模型在什么情况下会发起 **JSON 工具调用**，什么情况下会直接返回 UI 输出，并强调技能内容不是一次性全部暴露，而是采用类似 **“渐进式披露”** 的方式按需读取全文。这种机制对理解 **agent 工具调用协议、上下文管理、技能发现与触发** 非常有帮助。

从用途上看，这份 README 更像是一个 **实现原理/流程说明**，而不是面向普通用户的使用手册。它适合：
- 想研究 **Claude Code 源码** 的开发者
- 关注 **AI agent 工具调用** 的研究者
- 做 **prompt engineering** 的工程师
- 设计 **本地技能系统** 或类似插件机制的人
- 需要理解 **模型如何根据系统提示决定行动** 的技术读者
