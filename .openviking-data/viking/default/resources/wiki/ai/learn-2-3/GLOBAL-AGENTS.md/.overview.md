## Detailed Description

### GLOBAL-AGENTS.md GLOBAL-AGENTS.md

该文件是一份简短但核心明确的全局代理说明/规范文档，主要用于约束代理在面对不确定问题时的沟通与决策方式。文档的重点不是提供复杂流程或多层级章节，而是突出一个非常直接的原则：当信息模糊、上下文缺失、条件不足或无法判断时，不应自行推断、擅自执行或替用户做决定，而应主动向用户提问并等待确认。这种设计体现了“先澄清、后行动”的审慎工作流，有助于降低误操作和误判风险。

从用途上看，它更像是一份行为守则或交互约定，适合被嵌入到 AI 代理、自动化工作流、客服助手、文档生成器或其他需要处理用户指令的系统中。对于开发者而言，它可以作为统一决策边界的基础规则；对于协作者而言，它能够明确在信息不足时的沟通预期；对于系统实现者而言，它提供了一个简单而可执行的安全策略。整体关键词可概括为：用户确认、澄清问题、不确定性处理、决策约束、代理规范、交互式确认。

该文件虽然篇幅较短，但其价值在于提供了一个清晰的“停止并询问”机制，帮助系统保持可控性与透明度。若你希望构建一个不会轻易臆断、并且在关键节点严格依赖用户确认的代理行为模式，这份文档就是该目录中的核心参考。
