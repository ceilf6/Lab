## Detailed Description

### REP-AGENTS.md REP-AGENTS.md

这是 REP-AGENTS 项目的核心参考文档，定位为工程协作与开发规范的简要总览。文档以层级化方式组织，覆盖几个关键主题：**工程简要描述**、**开发规范**、**技术栈与工程结构**、**Git 规范**等，目的是为团队成员建立统一的代码开发与协作标准，减少不同成员在编码风格、目录理解、提交流程上的差异。

从内容定位上看，这份文档并不是业务说明书，而更接近于**项目指南 / 规范参考**。它适合在以下场景中使用：新成员入组时快速了解仓库规则；开发者在提交代码前确认工程约定；维护者在推进项目演进时统一规范口径。尤其值得注意的是，文档中关于 Git 提交的部分并非完全展开，而是进一步指向单独的 `git-role.md`，说明 Git 相关规范可能被拆分为更细的补充说明，以便集中管理提交流程、角色分工或提交约束。

总体而言，`REP-AGENTS.md` 承担的是**仓库入口文档**的角色：它帮助读者迅速把握 REP-AGENTS 的协作边界、结构组织与开发纪律，是理解整个仓库约定的第一站。
