## Detailed Description

### [1] `learn-2-3/`

`learn-2-3/` 是整个 `ai` 目录下的核心学习组织单元，主题围绕“输入 → Model → 输出”的认知框架展开，强调从数据进入模型、经过处理再到结果输出的完整链路。它的内容重点不在单个应用功能，而在基础理解：第二、第三层相关概念、网络数据流、代理服务器工作方式、抓包观察方法，以及如何把这些知识串成可操作的学习路径。

从目录结构看，这里没有独立的源码文件，更像是一个学习索引和协作入口。`README.md` 用来组织学习路线与目录导航；`GLOBAL-AGENTS.md` 提供全局行为规范，帮助统一仓库内的操作方式；`REP-AGENTS.md` 则偏向协作约定，适合多人协同阅读和维护。与此同时，`skill/`、`proxy-server/` 等子空间承担知识扩展或专题实践的作用，说明该目录并不是孤立文档，而是一个以基础认知为中心、逐步延展到实操主题的仓库化学习体系。

这个目录特别适合以下人群：刚开始学习网络原理、对代理与转发链路感到陌生、希望通过“观察输入和输出”来理解模型/系统行为的人；也适合希望整理知识图谱、建立目录式学习路径、或者在实践中边看抓包边理解协议层次的初学者与进阶入门者。
