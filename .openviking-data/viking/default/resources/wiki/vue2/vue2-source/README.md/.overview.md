### Vue_2_has_reached_End_of_Life.md Vue_2_has_reached_End_of_Life.md

This document is an end-of-life notice for Vue 2. It clearly states that the repository is inactive and that Vue 2 reached end of life on December 31, 2023. The page directs users to the actively maintained Vue 3 repository, `vuejs/core`, and provides migration-oriented guidance for those who need to move to the latest version.

The notice also clarifies that Vue 2 receives no new features, updates, or fixes, although it may still be available through common distribution channels. For users with compliance, legacy, or security needs, it points to Vue 2 NES support. The document is intended for existing Vue 2 users, maintainers, and developers evaluating framework choices or planning upgrades. It is especially important for teams managing legacy front-end applications or long-term maintenance strategies.
