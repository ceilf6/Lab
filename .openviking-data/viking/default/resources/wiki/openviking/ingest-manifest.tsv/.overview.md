## Detailed Description

### kind_path_target__7more_af2c7f56_1.md `kind_path_target__7more_af2c7f56_1.md`

这是一份 **仓库索引与摄取配置参考文档**，主要用于定义 OpenViking/GitNexus 的内容入库规则、目标映射和排除策略。文档整体按 **kind、target、exclude** 三个部分组织：其中 `kind` 说明 `file`、`dir`、`glob`、`exclude` 等规则的含义；`target` 说明内容可映射到 `viking://` URI 前缀或具体资源；随后列出大量仓库内高价值文档与目录的索引项，覆盖 README、AGENTS、docs、AI、CSS、HTML、Browser、NodeJS、React、Vue、TS、sandboxs 等主题，并为部分目录设置递归收录或排除。核心目标是通过 **白名单式 ingestion** 只收录精选知识节点，同时排除 `.git`、`node_modules`、`dist`、缓存和大型源码镜像，以保证检索质量与性能。该文档面向维护知识库、配置检索/索引流程的开发者或代理系统使用者，重点涉及 **Markdown、URI 映射、仓库结构、内容过滤、索引性能** 等关键词。
