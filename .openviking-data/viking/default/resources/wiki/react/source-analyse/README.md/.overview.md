## Detailed Description

### README.md README.md

该文件是一篇关于 Fiber 架构内部数据结构与状态更新流程的技术说明，核心目标是帮助读者理解 React 风格函数组件在 Fiber 体系中的运作方式。文档重点围绕几个关键概念展开：首先是 **虚拟树中 FiberNode 的链表结构**，说明 Fiber 并不是简单的树节点集合，而是通过链表方式串联形成可调度、可遍历的结构；其次是 **hook 对象形成的链表**，解释函数组件内部多个 hook 如何按顺序组织，并依赖调用顺序维持状态一致性；再者是 **`FiberNode.memoizedState` 作为 hook 链表头节点**，用于将组件当前的 hook 状态链挂载到 Fiber 节点上；最后是 **每个 hook 持有 `updateQueue`，并通过遍历 update 计算 state**，展示状态更新不是直接覆盖，而是通过队列逐步处理一系列更新动作，最终得到新的状态值。

从内容定位上看，这更像是一份面向源码阅读的笔记或原理讲解文档，而不是业务说明或使用手册。它适合前端开发者、React 源码学习者、Fiber 机制研究者，以及希望深入理解 Hooks 状态管理与更新调度流程的人阅读。文档中的关键词包括：**React Fiber、FiberNode、hook 链表、memoizedState、updateQueue、state 更新、函数组件、链表结构**。如果你正在分析 React 内部实现，或者想把“组件状态为什么这样更新”这件事讲清楚，这个文件会是本目录的核心入口。
