## Detailed Description

### core.md `core.md`

这份文档是一个关于 **CSS 继承性** 的简短说明，核心目标是帮助读者理解：并不是所有 CSS 属性都会从父元素传递到子元素，但与 **文字内容、字体排版** 相关的属性往往具有继承性。文档提到可以通过浏览器开发者工具或文档提示中的 **“Inherited from”** 来观察样式继承链，也说明了如何借助 **MDN 的“是否是继承属性”页面** 判断某个属性是否可继承。

文档的重点还包括：在实际开发中，像 `font-family` 这类字体属性通常适合直接写在 `body` 上，这样子元素会自动继承，减少重复设置；而 `font-variant` 等文字样式也常用于统一页面排版风格。整体内容偏基础，但对建立 CSS 样式层级、理解默认继承行为、以及提升前端调试效率很有帮助。适合正在学习 CSS、需要快速查阅继承规则、或希望规范化全局字体设置的开发者阅读。
