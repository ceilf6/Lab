# docs

这个目录是一个围绕前端工程实践、Vite 实验场、源码阅读与知识地图组织起来的文档集合，重点覆盖 **playground、sandbox、monorepo、HMR、ES modules、组件库构建、JavaScript 基础、前端研究导航** 等主题。它既包含仓库入门与使用说明，也包含对 Ant Design Mobile、JavaScript 对象模型等具体技术点的说明文档。整体适合前端开发者、学习者、源码阅读者、实验项目维护者，以及希望快速了解该知识库结构与使用方式的工程师。

## Detailed Description

### [1] `_README.md/`

这是一个基于 **Vite** 的实验性 playground 仓库入门说明，面向前端开发者、学习者以及需要搭建本地实验环境的工程师。内容围绕 **playground、sandbox、monorepo、HMR、ES modules、快速开始** 等关键词，说明如何在统一的项目结构中组织多个开发实验，并通过路由快速访问不同 sandbox。它强调模块化实验管理、自动索引、独立的 HTML/CSS/JS 结构，以及高频迭代能力，适合已经具备基础 npm、Vite 和现代 JavaScript 经验的读者。

### [2] `antd-mobile.md/`

这是关于 **Ant Design Mobile** 的结构说明文档，重点解释组件库如何组织源码、输出不同构建格式（如 **es / cjs / umd / bundle**），以及组件入口、全局聚合导出、`sideEffects`、`tree-shaking` 与 CSS 引入之间的关系。它不仅帮助理解组件库的打包与发布流程，也适合分析按需加载、样式生效策略和包体积优化的前端工程师、组件库维护者以及构建工具使用者。

### [3] `_README-CN.md/`

这是一个面向中文读者的项目指南目录，核心围绕一个基于 **Vite** 的实验场 Wiki 展开，主要用于学习、实践和复现前端工程模块。仓库内容重点覆盖 **Sandboxs、Agent、Monorepo** 等主题，并说明如何在 `sandboxs/<name>/` 下创建新实验、通过路由访问，以及使用 `npm install`、`npm run dev/build/preview` 完成本地开发与预览。它适合熟悉基础前端开发、Vite、Node.js、HMR 和模块化项目结构的学习者与开发者。

### [4] `WORKSPACE_MAP.md/`

这是一个面向前端技术研究与实验的工作区地图，集中整理了仓库中的目录导航、知识库分层、源码分析入口、自动化维护机制与实验资源。它覆盖 **前端原理、React/Fiber、Vue 源码、浏览器机制、网络、AI 探索**，以及可运行 demo 和辅助素材等内容，适合前端学习者、源码阅读者、实验项目维护者，以及希望快速理解这个知识库组织方式的使用者。整体上，它更像是前端实验室与学习地图的总入口。

### [5] `JS.md/`

这是一个关于 **JavaScript 基础概念** 的速览文档，核心聚焦于 **Function、函数对象、普通对象、对象创建关系、原型机制** 等关键主题。内容以概念说明为主，强调“Function → 产生函数对象 → 创造普通对象”这一对象模型链路，帮助读者理解 JavaScript 中“函数也是对象”的本质，以及函数、构造函数与对象之间的生成关系。适合正在学习 JavaScript 基础、需要梳理对象体系与类型关系、或希望快速检索相关概念的开发者。
