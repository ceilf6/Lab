## Detailed Description

### README-CN.md README-CN.md

这是一份面向开发者的项目指南文档，整体上介绍了一个基于 **Vite** 的实验场 Wiki。文档以“模块”“快速开始”“可用命令”“特性”等部分组织，帮助读者理解该仓库如何用于前端工程模块的学习、实践与复现。核心内容包括三个主要主题：**Sandboxs**、**Agent**、**Monorepo**，它们共同构成了项目的实验与组织方式。

在使用方式上，文档说明了常见的开发流程：先执行 `npm install` 安装依赖，再通过 `npm run dev` 启动开发服务器，默认地址为 `http://localhost:8030`；同时还提供 `npm run build` 和 `npm run preview`，分别用于构建生产产物与本地预览。对开发者而言，这些命令覆盖了从本地调试到发布检查的完整基础流程。

文档还强调了项目的实验管理机制：用户可以在 `sandboxs/<name>/` 下创建新的实验目录，并通过路由方式访问对应内容。项目支持自动扫描示例目录、HMR 热更新、独立的 HTML/CSS/JS 结构，以及首页示例列表展示，这使得它特别适合快速验证前端想法、拆分模块能力、测试页面交互和学习 Vite 生态中的工程化实践。

从仓库结构来看，这一目录本身更像是整个实验平台的入口说明与使用手册，而不是单一业务模块。它适合需要快速上手、理解项目约定、扩展实验案例或复现前端模块的开发者阅读。
