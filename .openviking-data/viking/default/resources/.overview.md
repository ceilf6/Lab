# resources

`resources` 是一个以 `wiki/` 为核心入口的知识型资源目录，整体更像一套面向前端与 Web 技术的“技术地图 + 学习导航 + 源码阅读索引”。它覆盖浏览器、Node.js、Vue、React、TypeScript、CSS、AI、知识检索等主题，适合前端开发者、源码阅读者、技术写作者、学习者，以及需要快速定位某个技术主题资料的人。这里的内容偏向文档驱动、知识沉淀和目录化组织，而不是单一业务应用或功能模块。

## Detailed Description

### wiki/ [1]

`wiki/` 是 `resources` 目录下的核心知识入口，定位为一个综合性的技术知识库与导航中心。它面向前端、Web、浏览器、Node.js、Vue、React、TypeScript、CSS、AI 与知识检索等多个方向，内容组织方式更接近“技术地图”“学习索引”和“源码阅读目录”。从结构上看，它把不同技术栈、实验区、文档区、图片资料以及工作流相关内容统一编排在一起，方便按主题快速跳转和逐步深入。

这个目录通常不承担单一业务逻辑，而是承担知识聚合、主题分类、资料沉淀与路径导航的职责。对于希望系统学习前端生态、查找某个技术专题资料、阅读源码分析、整理笔记或构建个人技术百科的人来说，它非常合适。整体风格偏文档化、目录化和参考型，强调可检索性与知识关联，而不是应用运行时功能。
