# Word MCP Server - 简化版说明

## 📝 简化内容

已将复杂的模块化项目简化为 **starter 风格**的简洁版本。

## 📁 现在的文件结构

```
wordMCP/
├── main_simple.py      # 主程序（约 400 行，包含所有功能）
├── pyproject.toml      # 项目配置（简化版）
├── README.md           # 使用文档
├── run_simple.sh       # 启动脚本
└── word/               # 文档目录（运行时自动创建）
```

## ✨ 保留的核心功能

### 9 个工具
1. ✅ `create_document` - 创建文档
2. ✅ `read_document` - 读取文档
3. ✅ `update_document` - 更新文档
4. ✅ `delete_document` - 删除文档
5. ✅ `list_documents` - 列出文档
6. ✅ `add_table` - 添加表格
7. ✅ `insert_image` - 插入图片
8. ✅ `format_text` - 格式化文本
9. ✅ `search_replace` - 搜索替换

### 1 个资源
- ✅ `file://documents` - 文档列表

### 1 个提示
- ✅ `help_prompt` - 帮助信息

## 🎯 简化对比

| 项目 | 之前（复杂版） | 现在（简化版） |
|------|------------|------------|

## 🚀 快速开始

```bash
# 1. 创建虚拟环境
python3 -m venv .venv

# 2. 激活
source .venv/bin/activate

# 3. 安装
pip install -e .

# 4. 运行
python main_simple.py
```

或使用启动脚本：
```bash
./run_simple.sh
```

## 📖 使用示例

### 创建并编辑文档
```python
# 创建
create_document(filename="test.docx", title="测试", content="内容")

# 读取
read_document(filename="test.docx")

# 更新
update_document(filename="test.docx", action="append", content="新段落")

# 添加表格
add_table(
    filename="test.docx",
    table_data=[["列1", "列2"], ["数据1", "数据2"]]
)
```
