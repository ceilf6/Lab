浏览器是一个元素一个元素依次渲染，顺序是按照页面文档的树形目录结构进行

html

|- head

|-meta

|-title

|- body

|-img

|-p

渲染每个元素的前提条件：该元素的所有 CSS 属性必须有值

Element 的 computed 中 show all

这个计算过程，叫做属性值计算过程

（深度优先：父 → 子 → 孙 …）

## 4. 使用默认值

对仍然没有值的属性，使用默认值

MDN 查 Initial value

例如背景颜色默认是透明

background-color: transparent

(0,0,0,0) 最后一个0是透明通道

[]()

```html
    <!-- 
    浏览器对 a 元素已经有声明值了，所以在 1 就确定了值，不会到 3继承
    -->
    <div class="container">
        <a href="">a</a>
        <p>p</p>
    </div>
```

```html
a {
    /* 希望继承，但是继承是在第三步的，小于前面的声明 */
    /* 那么就声明继承 */
    color: inherit;
}
```

## 特殊的两个CSS取值

1. inherit 除了能声明继承 - 提前继承
    
    还能继承无法继承的属性
    
2. initial 初始值
将该属性设置为默认值
